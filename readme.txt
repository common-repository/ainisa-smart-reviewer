Smart AI comment/review writer for your wordpress website

== Description ==

Sometimes you need to add some reviews to your website but you don't have time to think about reviews and add them. For this purpose we have created AI Smart Reviewer plugin for your wordpress website. You can add unlimited reviews for your posts. A new user will be created for each review. All reviews are generated by AI and we tried to make them like human created reviews.

== Frequently Asked Questions ==

= How to configure plugin ? =

* Go to Ai Smart Reviewer page in your Wordpress admin panel
* Click to Settings tab
* In settings tab change your configuration

= How to add reviews ? =
* Go to Ai Smart Reviewer > Add New Review page
* Click to Get Posts button. You will get list of your posts
* Below select the post for which you want to create review and click Get AI Review button
* Click Save review button in order to save the review
